# Packets

This folder holds **reference/authoring** copies of AI Planner guidance prompts and copy bundles (e.g. `CURSOR_PROMPT_*.txt`, `CURSOR_PACKET_*.md`). They were inlined into the app for runtime use.

**Live copy:** The strings shown in the app live in **`components/AiPlannerGuidance.js`**. Edit that file to change what users see. These packet files are for history and reference only; they are not loaded at runtime.
